### О проекте
short_info_AIDA - это GUI парсер HTM файлов программы AIDA64.
Приложение помогает автомотизировать процесс копирования данных компьютеров для последующей вставки в Excel таблицу.

## Технологии
- Python
- Kivy
- KivyMD
- BeautifulSoup4
